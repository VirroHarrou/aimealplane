# 🥗 AI Meal Planner
AI Meal Planner — это высокопроизводительное Flutter-приложение для стратегического планирования питания. Система использует возможности генеративного ИИ для составления рационов, адаптированных под конкретные физические показатели и цели пользователя (похудение, набор массы, поддержание здоровья).

## 🚀 Ключевые особенности
- Динамическое создание планов питания через интеграцию с Gemini 2.5 Flash.
- Полное управление жизненным циклом данных (профиль, планы, ошибки) с помощью Riverpod.
- Хранение данных firestore + shared_prefrences 

## 🛠 Технологический стек
- Framework: Flutter (Dart).
- State Management: Riverpod (StateNotifier).
- Database: Firestore.
- AI Engine: Google Generative AI (Gemini API).
- Environment: flutter_dotenv для безопасного хранения ключей.

## 🏗 Архитектура системы
Проект построен на принципах разделения ответственности:

- StorageService: Управляет транзакциями записи профиля и извлечением закэшированных JSON-ответов с использованием firestore.

- GeminiService: Инкапсулирует логику общения с ИИ. Использует строгий системный промпт для получения валидного JSON без лишнего текста.

- UserProfileNotifier: Источник данных пользователя. Инициализирует загрузку профиля из БД при старте приложения.

- MealPlanNotifier: Слушает изменения профиля и автоматически синхронизирует UI с кэшем или инициирует генерацию нового плана.

## ⚙️ Установка и запуск
Клонируйте репозиторий.

Запуск:

```bash
flutter run \
  --dart-define=GEMINI_API_KEY=ваш_ключ \
  --dart-define=AF_DEV_KEY=ваш_ключ \
  --dart-define=AF_APP_ID=ваш_id \
  --dart-define=APPHUD_KEY=ваш_ключ \
  --dart-define=APPMETRICA_KEY=ваш_ключ
```
## 🛡 Обработка исключений

Система спроектирована с учетом нестабильного соединения:

При отсутствии интернета и пустом кэше приложение выдает информативное сообщение через AsyncValue.error.

При некорректном ответе от ИИ (ошибки парсинга JSON) предусмотрен защитный слой в UI, отображающий сырые данные для отладки.
